O que é HTML?

HyperText Markup Language - É uma linguagem de marcação de Hipertexto, é essencial para a programação web. Ele permite criar novos websites e a inserção de componentes.

Importante saber que o HTML não é uma linguagem de programação, é mais uma linguagem de marcação.

Para quem está entrando no mundo da prorgamação web é extremamente importante aprender.

Tags aprendida nas aulas

h1 até h6 - São usadas para criar títulos em páginas web. Cada tag tem a sua
importância, por númeração, começando do maior ao menor, na ordem de 1 a 6.
p - Para criar parágrafos.
mark - Marca um texto igual ao marca texto.
small - Deixa o texto menor.
i - Deixa o texto em itálico.
u - Deixa o texto sublinado.
strong - Torna o texto importante e aparentemente em negrito.
ol - Cria uma lista ordenada.
ul - Cria uma lista desordenada.
li - Representa o item de uma lista.
a - Cria vinculo com outro documento.
sub - Cria um texto com letras menores abaixo da linha do conteúdo.
sup - Cria um texto com letras menores acima da linha do conteúdo.
blockquote - Cria a citação dentro de um bloco.

Tags extras

font - Usado para configurar o tipo, cor e tamanho das letras de uma página.
del - Representa determinada parte de um texto que foi excluída de um documento.
abbr - Representa uma abreviação e também fornece de forna opcional a descrição completa
